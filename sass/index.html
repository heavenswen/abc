• SASS css预处理器

    sass有两种后缀名文件：一种后缀名为sass，不使用大括号和分号；另一种就是我们这里使用的scss文件，这种和我们平时写的css文件格式差不多，使用大括号和分号。而本教程中所说的所有sass文件都指后缀名为scss的文件。在此也建议使用后缀名为scss的文件，以避免sass后缀名的严格格式要求报错。

    注释：

    sass支持‘//’和‘/**/’的方法，但单行注释不会被编译出来

    变量：

    以‘$’为声明变量，在变量后面添加!default为声明一个默认变量，可以通过在!default前声明变量来覆盖变量,在变量值后面添加!global为全局变量(sass3.4后)

    变量还可以用在css选择器中 .class-#{$var}{//code...},通过#{$var}来调用

    变量也可以作为样式 名使用 {border-#{$var}:1px solid #ccc;font:#{$var}/#{$var}}

    数组：
    • 一维:$arr:5px 0px 0px 10px;
    • 二维:$arr:11px 10px,20px 11px;or $arr:(11px 10px)(12px 30px) 用‘,’或“()”分隔

      • 使用{padding:nth($arr,1)}

    • 对象：$obj:(h1:18px,h2:16px)

      • 可以使用@each $key ,$value in $obj{#{$key}:font-size:$value }循环输出

    嵌套：

    sass 不仅可以选择器嵌套，还可以进行样式的嵌套

    .class{
    border{
    style:solid
    color:#39f
    }
    }
    @at-root (without: ...)和@at-root (with: ...) 跳出

    默认@at-root只会跳出选择器嵌套，而不能跳出@media或@support，如果要跳出这两种，则需使用@at-root (without: media)，@at-root (without: support)。这个语法的关键词有四个：all（表示所有），rule（表示常规css），media（表示media），support（表示support，因为@support目前还无法广泛使用，所以在此不表）。我们默认的@at-root其实就是@at-root (without:rule)。

    混合mixin 类似function之于js
    @mixin name($color:#39f){
    border:1px solid $color
    }

    @content可以用来解决css3的@media等带来的问题。它可以使@mixin接受一整块样式，接受的样式从@content开始。

    @mixin max-screen($res){
    @media only screen and ( max-width: $res )
    {
    @content;
    }
    }
    @include max-screen(480px) {
    body { color: red }
    }
    //css style
    //-------------------------------
    @media only screen and (max-width: 480px) {
    body { color: red }
    }

    PS：@mixin通过@include调用后解析出来的样式是以拷贝形式存在的，而下面的继承则是以联合声明的方式存在的，所以从3.2.0版本以后，建议传递参数的用@mixin，而非传递参数类的使用下面的继承%
